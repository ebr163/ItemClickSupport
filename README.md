# ItemClickSupport
Add click listener on RecyclerView

Add it in your root build.gradle at the end of repositories:
```gradle
allprojects {
		repositories {
			maven { url "https://jitpack.io" }
		}
	}
```
Add the dependency
```gradle
dependencies {
	        compile 'com.github.ebr163:ItemClickSupport:1.0.0'
	}
```
